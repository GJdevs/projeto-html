Tudo Sobre head em HTML

A tag head em HTML é um elemento que contém metadados sobre o documento, ou seja, informações que não são exibidas diretamente na página web, mas são essenciais para seu funcionamento e comportamento.

Dentro da tag, você pode incluir:

  • Metadados: como o título da página (tittle), descrições, palavras-chave, autor, etc.
  • Links para recursos externos: como arquivos de estilo CSS (links) e fontes externas.
  • Scripts: como arquivos JavaScript (script) que podem ser carregados antes ou depois do conteúdo da página
  • Charset: a codificação de caracteres usada no documento

A tag ajuda a configurar a página e otimizar o carregamento e a acessibilidade. Ela geralmente aparece no início do código HTML, antes da tag body, que contém o conteúdo visível.

Tudo Sobre body em HTML

A tag em HTML contém o conteúdo visível de uma página web, como texto, imagens, links e outros elementos. Tudo que é exibido ao usuário na página está dentro dessa tag.

Tudo Sobre Links em HTML

A tag link em HTML é usada para vincular documentos externos ao HTML, como arquivos de estilo CSS, fontes externas e ícones. Normalmente, é colocada dentro da tag head e não exibe conteúdo na página.

  • rel: Define o tipo de relação (como "stylesheet" para arquivos CSS).
  • href: Especifica o caminho do arquivo vinculado.
